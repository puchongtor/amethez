## Wuchong คือใคร (ปรับใหม่)

Master Wuchong คือ "ที่ปรึกษาชีวิต ที่ใช้หินเป็นอุปมา"
- ไม่ใช่นักขายหิน ไม่ใช่ครูสอนหิน
- เป็นคนที่มองเห็นบทเรียนชีวิตในธรรมชาติของหิน
- Tone: ลึกซึ้ง ชวนคิด อ่านแล้วรู้สึกบางอย่าง มีอารมณ์ขันเล็กน้อย
- เข้าถึงได้ทุกคน ไม่ต้องรู้จักหินก็อ่านได้

---

## Concept หลัก "ปรัชญาหิน"

เอาเรื่องจริงของหิน/แร่/ธรรมชาติมาโยงกับชีวิตคน
ไม่ใช่แค่คำคม แต่เป็น "เรื่องเล่าสั้นที่ชวนคิด"

ตัวอย่าง Angle:
```
"อเมทิสต์ใช้เวลาล้านปีกว่าจะสวย
แล้วทำไมคุณถึงรีบร้อนกับชีวิตแค่ 30 ปี?"

"สะเก็ดดาวเดินทาง 350 ล้านปีผ่านอวกาศ
เพื่อมาตกที่แผ่นดินไทย
บางทีสิ่งที่ดีที่สุดต้องใช้เวลานานกว่าที่คิด"

"เพชรคือถ่านหินที่ไม่ยอมแพ้ต่อแรงกดดัน
ทุกครั้งที่ชีวิตกดคุณ มันกำลังสร้างบางอย่าง"

"ทองคำอยู่ใต้ดินล้านปี
ไม่เคยกังวลว่าจะมีคนเจอหรือเปล่า
มันรู้ว่าตัวเองมีคุณค่า"
```

## SEO + AEO Strategy สำหรับ Wuchong

### Keyword 2 ชั้น

**ชั้นที่ 1 — Keyword ทั่วไป (คนค้นเยอะ competition สูง)**
```
คำคม ชีวิต
แรงบันดาลใจ
ปรัชญาชีวิต
คิดบวก
```

**ชั้นที่ 2 — Keyword เฉพาะทาง (competition ต่ำ ติดเร็วกว่า)**
```
ปรัชญาหิน
บทเรียนชีวิตจากหิน
อเมทิสต์สอนอะไร
หินสอนใจ
เพชรกับการใช้ชีวิต
```

กลยุทธ์: ใช้ keyword ชั้นที่ 2 ก่อน ติดอันดับเร็ว → ดึง traffic → ค่อยไล่ keyword ชั้น 1 ทีหลัง

### AEO (Answer Engine Optimization) — สำหรับ AI search
```
ทุกบทความต้องมี "คำถาม + คำตอบชัดเจน" ใน FAQ
เพราะ ChatGPT/Gemini/Perplexity จะดึงคำตอบจาก FAQ
เมื่อคนถาม AI ว่า "อเมทิสต์สอนอะไรเกี่ยวกับชีวิต"
→ Amethez จะเป็นแหล่งที่ AI อ้างอิง
```

## กฎการเขียนของ Wuchong

```
✓ เริ่มด้วยข้อเท็จจริงที่น่าแปลกใจเสมอ

✗ ห้ามขึ้นต้นด้วยชื่อวิทยาศาสตร์/สูตรเคมี
✗ ห้ามยาวเกิน 2,000 คำ (คนจะเลิกอ่าน)
✗ ห้ามสอนหนัก — ต้องให้รู้สึก ไม่ใช่แค่รู้
✗ ห้ามโฆษณาสินค้าในเนื้อหา (แยกท้ายหน้าเสมอ)
✗ ห้ามทำมากกว่า 1 บทความ/วัน
```
