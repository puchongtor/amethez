# คำสั่งสำหรับ Claude Code — ระบบคำคม "ปรัชญาหินรายวัน"
> 1 คำคม/วัน ฝังอยู่ใน sidebar หรือ block พิเศษของบทความ Wuchong

---

## ภาพรวมระบบ

```
บทความ Wuchong (1,500-2,000 คำ)
        +
[WISDOM BLOCK] — คำคมปรัชญาหินประจำวัน (1 ข้อ)
        ↓
แสดงใน sidebar หรือ block พิเศษในบทความ
เปลี่ยนอัตโนมัติทุกวัน ตาม date
ไม่ต้องทำใหม่ทุกวัน — สร้างสต็อกล่วงหน้าครบ 365 ข้อ
```

## โครงสร้าง Wisdom Block ในหน้าบทความ

```html
<aside class="wisdom-block">
  <div class="wisdom-date">วันนี้ Wuchong ฝากไว้</div>
  <div class="wisdom-stone-icon">💎</div>
  <blockquote class="wisdom-quote">
    "[คำคมหลัก 1-2 บรรทัด]"
  </blockquote>
  <p class="wisdom-stone-name">— [ชื่อหิน] สอนเรา</p>
  <a href="/wuchong/[slug-บทความเต็ม]" class="wisdom-read-more">
    อ่านเรื่องเต็ม →
  </a>
</aside>
```

**Design spec:**
- พื้นหลัง: gradient ม่วงเข้ม (#1a0d2e → #2d1b69)
- ตัวหนังสือ: ขาว/ทอง (#c9a84c)
- border: เส้นทอง 1px
- icon หินเปลี่ยนตามหินที่อ้างอิง
- มี link ไปบทความ Wuchong เต็มๆ เสมอ

**ตำแหน่งใน layout:**
- Desktop: Sidebar ขวามือ sticky ติดขณะ scroll
- Mobile: Block พิเศษ แทรกหลัง H2 แรกของบทความ

## วิธีที่ระบบทำงาน (js/wisdom.js)

```javascript
// อ่านวันที่ปัจจุบัน
const today = new Date().toISOString().split('T')[0];

// ดึง wisdom.json
fetch('/data/wisdom.json')
  .then(r => r.json())
  .then(data => {
    // หาคำคมของวันนี้
    const todayWisdom = data.wisdom.find(w => w.date === today)
      || data.wisdom[data.wisdom.length - 1]; // fallback คำคมล่าสุด

    // render Wisdom Block
    document.querySelector('.wisdom-quote').textContent = todayWisdom.quote;
    document.querySelector('.wisdom-stone-icon').textContent = todayWisdom.stone_icon;
    document.querySelector('.wisdom-stone-name').textContent =
      `— ${todayWisdom.stone} สอนเรา`;
    document.querySelector('.wisdom-read-more').href =
      `/wuchong/${todayWisdom.article_slug}`;
  });
```

### หมวด 1 — ความอดทน/เวลา (ม.ค. วันที่ 1-31)
```
1.  อเมทิสต์ล้านปีกว่าจะสวย คุณรีบร้อนทำไมกับชีวิต 30 ปี?
2.  หินไม่เคยรีบ แต่ก็ไม่เคยหยุด
3.  ทุกวันที่ผ่านไปคือชั้นหนึ่งของหินอ่อน
4.  ถ้ำหินงอกใช้ร้อยปีต่อ 1 เซนติเมตร ผลลัพธ์ที่ดีต้องการเวลา
5.  แกรนิตเย็นตัวช้าๆ ลึกใต้ดิน สิ่งที่แข็งแกร่งมักเกิดอย่างเงียบๆ
6.  หินปูนสะสมทีละเม็ดทราย มหาสมุทรเก่าก็เริ่มจากหยดน้ำ
7.  อำพัน 385 ล้านปียังคงใสสะอาด ความอดทนรักษาทุกอย่างไว้
8.  เพิร์ลใช้เวลาปีกว่าจะกลมสมบูรณ์ ความสมบูรณ์ไม่มีทางลัด
9.  หินทรายถูกน้ำกัดเซาะพันปีจนกลายเป็นแกรนด์แคนยอน ความต่อเนื่องสร้างสิ่งยิ่งใหญ่
10. ซากดึกดำบรรพ์ต้องรอหลายล้านปีกว่าจะถูกพบ บางอย่างต้องรอจังหวะที่ใช่
11. หินที่ร้อนที่สุดอยู่ลึกที่สุด ความลึกของประสบการณ์มาจากความเข้มข้นของชีวิต
12. ทราย + เวลา + แรงดัน = หินทราย ทุกสิ่งต้องผ่านกระบวนการ
13. ผลึกควอตซ์เติบโต 1 มิลลิเมตรต่อปี ความเติบโตที่แท้จริงช้าและมั่นคง
14. ดินเหนียวต้องผ่านไฟ 1,200 องศาจึงกลายเป็นเครื่องกระเบื้องถาวร ความแข็งแกร่งต้องผ่านความร้อน
15. หินที่อยู่ในแม่น้ำนานที่สุดมักเกลี้ยงที่สุด ผ่านมากเผชิญมาก จึงเรียบได้
16. ฟอสซิลต้องใช้เวลาหมื่นปีกว่าจะสมบูรณ์ สิ่งที่ยั่งยืนใช้เวลาสร้าง
17. น้ำแข็งที่กดทับหินล้านปีสร้างหุบเขาได้ แรงเล็กๆ สม่ำเสมอชนะแรงใหญ่ครั้งเดียว
18. หินมีอายุมากกว่ามนุษย์พันล้านเท่า แต่เราเลือกได้ว่าจะทำอะไรกับเวลาที่มี
19. เพิร์ลไม่ได้งามเพราะรีบ แต่เพราะทนอยู่กับความเจ็บปวดนานพอ
20. แกรนิตในหลุมศพโรมันโบราณยังคงสวยงามหลัง 2,000 ปี ของดีทนทานกาลเวลา
21. หินอ่อนจากกรีซโบราณยังขาวบริสุทธิ์หลังพันปี คุณภาพแท้ไม่ซีดจาง
22. ทองคำที่ขุดพบในอียิปต์โบราณยังเป็นทองคำ คุณค่าที่แท้จริงไม่ขึ้นกับเวลา
23. หินงอก 1 เมตรในถ้ำใช้เวลา 4,000-10,000 ปี บางสิ่งงามเพราะใช้เวลา
24. ไดมอนด์ต้องอยู่ใต้ดินนาน 1-3 พันล้านปีก่อนจะขึ้นสู่ผิวโลก ทุกอย่างมีเวลาของมัน
25. หินภูเขาไฟเย็นตัวภายในชั่วโมง แต่สร้างเกาะได้ทั้งเกาะ บางครั้งสิ่งยิ่งใหญ่เกิดเร็วด้วย
26. ทราย 1 กำมือมีหินหลายชนิดรวมกัน ความหลากหลายทำให้สมดุล
27. แม่น้ำโขงใช้เวลาล้านปีกัดเซาะหินจนได้ช่องแคบ ทำสิ่งเดิมซ้ำๆ ได้ผลเสมอ
28. อำพันที่มีแมลงโบราณข้างในมีค่ากว่าอำพันเปล่า ประสบการณ์เพิ่มคุณค่า
29. ดีบุกที่ใช้ในสมัยโรมันยังคงอยู่ในเปลือกโลก ทุกสิ่งที่เคยมีอยู่ยังอยู่ที่ไหนสักแห่ง
30. หินเหล็กไฟที่มนุษย์โบราณใช้จุดไฟยังมีอยู่ สิ่งที่มีประโยชน์ไม่มีวันล้าสมัย
31. ผลึกที่เติบโตช้าที่สุดมักสวยงามและสมบูรณ์ที่สุด ความรีบร้อนทำให้ไม่สมบูรณ์
```

### หมวด 2 — ความแข็งแกร่ง/แรงกดดัน (ก.พ. วันที่ 32-59)
```
32. เพชรคือถ่านหินที่ไม่ยอมแพ้ต่อแรงกดดัน

34. ซิทรินสีทองเข้มเกิดจากความร้อนสูง ความร้อนเปลี่ยนเราได้เสมอ

36. หินที่ถูกน้ำกัดเซาะจนเรียบคือหินที่ผ่านอะไรมามากที่สุด

46. เพชรตัดกระจกได้แต่กระจกตัดเพชรไม่ได้ รู้จักพลังของตัวเอง
47. หินแม่น้ำที่ผ่านน้ำมาพันปีไม่เคยหยุดเกลี้ยง ความสม่ำเสมอสร้างความสมบูรณ์
48. เพชรที่ตัดแล้วแข็งกว่าก่อนตัด การเผชิญกับสิ่งท้าทายทำให้แข็งแกร่งขึ้น
49. หินอัคนีเกิดจากความร้อนที่สูงที่สุดในโลก ของแข็งแกร่งที่สุดผ่านความร้อนสูงที่สุด
50. ทองคำบริสุทธิ์ต้องถูกเผาหลายครั้งจึงจะสะอาด การทดสอบซ้ำๆ คือการชำระ
51. แร่ที่แข็งที่สุดมักพบในสภาพแวดล้อมที่รุนแรงที่สุด
52. หินแปรสภาพเกิดจากหินอื่นที่ผ่านความร้อนและแรงดัน เราทุกคนกำลังแปรสภาพอยู่
53. แผ่นทวีปชนกัน 50 ล้านปีจนเกิดเทือกเขาหิมาลัย การชนกันครั้งใหญ่สร้างสิ่งยิ่งใหญ่
54. คาร์เนเลียนเป็นหินของนักรบโบราณเพราะทนทาน ความแข็งแกร่งคือเกราะที่ดีที่สุด
55. หินที่ถูกตัดแต่งสวยงามกว่าหินก้อนเดิม การเผชิญกับสิ่งที่ท้าทายเผยศักยภาพ
56. เพชรไม่สามารถตัดเพชรด้วยกันได้ง่ายๆ แม้ของแข็งที่สุดก็ยังต้องการกลยุทธ์
```
