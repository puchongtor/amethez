## 3 สิ่งที่ต้องเพิ่ม (Priority สูงสุด)

### เพิ่มที่ 1 — Emotional Hook ในดวงรายวันแต่ละราศี

**ปัญหาเดิม:** ดวงรายวันบอกแค่ "ดาวเด่น + ระดับดาว + หมวด" คนอ่านแล้วไม่รู้สึกอะไร

**วิธีแก้:** เพิ่ม 2-3 ประโยคในแต่ละราศีที่พูดถึงความรู้สึก + สิ่งที่ควรทำ

**Format ใหม่ต่อ 1 ราศี:**
```
[ชื่อราศี] — [วันที่]
[ดาวเด่น] · [ระดับดาว ⭐] · [หมวดดวง]

"[Emotional Hook 1-2 ประโยค — พูดถึงความรู้สึกที่คนน่าจะเป็นวันนี้]
[บทเรียน/สิ่งที่ควรระวัง/โอกาสที่มา]"

💎 หินเสริมวันนี้: [ชื่อหิน]
🔮 Ritual: [สิ่งที่ทำได้ทันทีใน 1 นาที]
→ [ลิงก์ Shopee หินนั้น]
```

**ตัวอย่างจริง:**
```
ราศีเมษ — วันพุธที่ 22 กรกฎาคม 2569
☀️ พระอาทิตย์ · ⭐⭐⭐⭐ · การงานดี

"วันนี้คุณอาจรู้สึกเหนื่อยกับสิ่งที่ค้างคาใจมานาน
แต่ดาวพฤหัสกำลังส่งพลังมาเปิดทางให้คุณอยู่"

💎 หินเสริมวันนี้: ซิทริน
🔮 Ritual: ถือซิทรินในกำมือขวา นึกถึงสิ่งที่อยากให้สำเร็จ 3 วินาที ก่อนออกจากบ้าน
→ [Shopee: ซิทริน]
```

### เพิ่มที่ 3 — ค้นหาดาวประจำตัวจากวันเกิด

**ปัญหาเดิม:** หน้า "ดวงชะตากับหิน 9 ดาว" สวยมาก แต่คนต้องรู้ว่าตัวเองเป็นดาวอะไรก่อน

**วิธีแก้:** เพิ่ม mini calculator ให้กรอกวันเกิด → ระบบคำนวณดาวเจ้าชะตา → แสดงหินที่เหมาะ

```html
<section class="birth-stone-finder">
  <h3>🔮 ค้นหาดาวเจ้าชะตาของคุณ</h3>
  <p>กรอกวันเกิดเพื่อดูว่าดาวไหนเป็นเจ้าชะตา และหินไหนเหมาะกับคุณที่สุด</p>

  <input type="date" id="birthdate" placeholder="วัน/เดือน/ปีเกิด">
  <button onclick="findBirthStar()">ค้นหาดาวของฉัน ✨</button>

  <div id="result" class="birth-result">
    <!-- แสดงผล: ดาวเจ้าชะตา + อิทธิพล + หิน 3 ชนิด + ลิงก์ Shopee -->
  </div>
</section>
```

**Logic การคำนวณ (วันเกิด → ดาวเจ้าชะตา โหราศาสตร์ไทย):**
```javascript
function findBirthStar(birthdate) {
  const day = new Date(birthdate).getDay();
  const stars = {
    0: { name: "พระอาทิตย์", stones: ["ทับทิม", "Red Carnelian", "Sunstone"] },
    1: { name: "พระจันทร์", stones: ["ไข่มุก", "Moonstone", "Selenite"] },
    2: { name: "พระอังคาร", stones: ["เลือดหมู", "Bloodstone", "Garnet"] },
    3: { name: "พระพุธ", stones: ["มรกต", "Green Aventurine", "Amazonite"] },
    4: { name: "พระพฤหัสบดี", stones: ["บุษราคัม", "Citrine", "Pyrite"] },
    5: { name: "พระศุกร์", stones: ["เพชร", "Rose Quartz", "Morganite"] },
    6: { name: "พระเสาร์", stones: ["นิล", "Amethyst", "Sodalite"] }
  };
  return stars[day];
}
```

### Topic List 40 บทความ (สำหรับ Claude Code ทำต่อ)

#### กลุ่ม 1 — พระพิฆเนศ + หิน (สำคัญที่สุด เป็นสินค้าเด่น)
```
1.  พระพิฆเนศชอบหินอะไร — ไหว้ถูกหินได้พรเร็วกว่า
2.  วันพฤหัสไหว้พระพิฆเนศด้วยหิน 3 ชนิดนี้ให้ได้พรด้านการงาน
3.  พระพิฆเนศสีอะไร เสริมพลังด้านไหน — หินที่เหมาะกับแต่ละรูป
4.  พิธีบวงสรวงพระพิฆเนศแบบสายหิน ของที่ต้องมี 7 อย่าง
5.  พระพิฆเนศงาหัก vs งาครบ พลังงานต่างกันอย่างไร หินที่เสริมแต่ละแบบ
6.  วางพระพิฆเนศทิศไหนดีที่สุด + หินที่วางคู่กันได้
7.  พระพิฆเนศ 32 ปาง แต่ละปางใช้หินอะไรเสริม
8.  สีดอกไม้ไหว้พระพิฆเนศ + หินสีที่ match กัน
9.  ทำไมพระพิฆเนศถึงชอบของหวาน — ความเชื่อและหินที่เสริมพลังนี้
10. คาถาพระพิฆเนศ + หินที่ถือขณะท่อง ได้ผลจริงไหม
```
