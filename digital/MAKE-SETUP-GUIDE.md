## ⚙️ ขั้นตอนติดตั้ง (ทำตามทีละขั้น)

### STEP 1 — สมัคร SlipOK และเชื่อมบัญชีธนาคาร

1. ไปที่ **slipok.com** → สมัครสมาชิก
2. เพิ่มบัญชีธนาคารที่จะรับเงิน (เบอร์ PromptPay ที่ใช้ในเว็บ)
3. ไปที่ **API Settings** → คัดลอก:
   - `API Key`
   - `Branch ID` (รหัสสาขา)
4. เก็บ 2 ค่านี้ไว้ใช้ใน Make.com

---

### STEP 2 — เตรียมไฟล์ E-Book บน Google Drive

1. แปลง `ebook-moldavite-wealth-content.html` เป็น PDF
   - เปิดไฟล์ในเบราว์เซอร์ → กดปุ่ม "บันทึกเป็น PDF" → Save
2. อัปโหลด PDF ขึ้น **Google Drive**
3. คลิกขวาไฟล์ → **แชร์** → "ทุกคนที่มีลิงก์" → คัดลอกลิงก์
4. เก็บลิงก์นี้ไว้ (จะใส่ใน Make.com)

> 💡 **Bonus 3 อย่าง** (Wallpaper, Cards, Tracker) — รวมเป็น PDF เดียวกัน หรือทำ ZIP ก็ได้

---

### STEP 3 — สร้าง Google Sheet เก็บออเดอร์

สร้าง Sheet ชื่อ **"Amethez Orders"** มี column:

| A | B | C | D | E | F | G |
|---|---|---|---|---|---|---|
| timestamp | email | name | product_id | amount | token | expires |

Make.com จะเขียนข้อมูลลงนี้อัตโนมัติทุกครั้งที่ขายได้

---

### STEP 4 — Import Scenario เข้า Make.com

1. ไปที่ **make.com** → Create new Scenario
2. กดเมนู (•••) → **Import Blueprint**
3. เลือกไฟล์ `make-blueprint-auto.json` (อยู่ในโฟลเดอร์นี้)
4. ใส่ค่าที่ระบบถาม:
   - SlipOK API Key + Branch ID
   - Google Drive link ของ E-Book
   - Gmail account
   - Google Sheet ID
   - Line Notify token (ถ้าต้องการแจ้งเตือน)
5. **คัดลอก Webhook URL** ที่ได้

---

### STEP 5 — ใส่ Webhook URL ลงเว็บ

เปิดไฟล์ `digital/moldavite-wealth.html` หาบรรทัด:

```javascript
const CFG = {
  promptpay: '0812345678',           // ← ใส่เบอร์ PromptPay จริง
  amount: 299,
  webhook: 'https://hook.eu1.make.com/REPLACE_WEBHOOK_ID',  // ← วาง Webhook URL ที่ได้จาก Make
  ...
};
```

แก้ 2 จุด: `promptpay` และ `webhook` → save → push GitHub → เสร็จ!

## 📞 ถ้าติดปัญหา

- Make.com ไม่ทำงาน → เช็คว่า Scenario เปิด ON อยู่ (มุมซ้ายล่าง)
- อีเมลไม่ส่ง → เช็ค Gmail connection ใน Make
- SlipOK error → เช็ค API Key + บัญชีธนาคารผูกถูกไหม
- สลิปผ่านแต่ไม่ส่งเมล → ดู Router เงื่อนไขใน Make
